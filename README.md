## O que é?

O **Caça aos Bugs** é um desafio destinado aos alunos presentes na aula do dia 18/06, organizada no componente curricular de Projeto de Aprendizagem Extensionista (PAC).

Basta baixar o código e abri-lo na IDE DevC++, encontrar os erros (16) e solucioná-los. Tente resolver dentro do tempo previsto e assim que terminar avise em voz alta.

**Não se preocupe! Todos estamos disponíveis para ajudar vocês!**


---

## Objetivos

* Incentivar o raciocínio lógico;
* Aumentar a compreensão de cada parte de um código C;
* Inserir o aluno em um contexto real de trabalho.
